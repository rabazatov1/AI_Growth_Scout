# Источники данных — AI Growth Scout

## 1. Stepik API (живые данные)

- **URL:** https://stepik.org/api/courses
- **Тип:** Публичный REST API (без авторизации)
- **Что получаем:** Курсы по программированию, ML, анализу данных и смежным темам
- **Язык:** Русский и английский
- **Фильтрация:** Поиск по ключевым словам, сортировка бесплатных курсов первыми
- **Ограничения:** Нет информации о дедлайнах (курсы открыты постоянно), нет фильтра по уровню через API (определяем эвристически из названия/описания)
- **Дата подключения:** 2026-04-23

## 2. Кураторский датасет (статические данные)

- **Файл:** `data/opportunities_curated.json`
- **Тип:** Ручная подборка
- **Дата сбора:** 2026-04-24
- **Количество записей:** 16
- **Что включено:**
  - Стажировки: Young&&Yandex, Сбер, Т-Старт, Selectel Career Wave, Газпром нефть, YADRO Импульс, MTS Tech
  - Курсы: mlcourse.ai (ODS.ai), Hexlet (Frontend, Python), Karpov.Courses ML, Нетология ML
  - Хакатоны: MOEX AI Hackathon, Город IT:HACK 2026
  - Программы: ШАД Яндекса, Яндекс Лицей
- **Ограничения:** Дедлайны могут устаревать, ссылки требуют периодической проверки

## 3. Groq API / LLM

- **URL:** https://api.groq.com/openai/v1
- **Тип:** Бесплатный API (OpenAI-совместимый)
- **Назначение:** Разбор запросов пользователя (intent parsing) и генерация связного текстового ответа
- **Модель:** llama-3.3-70b-versatile (Llama 3.3 от Meta, хостинг Groq)
- **Требует:** OPENAI_API_KEY в файле .env (ключ с console.groq.com)
- **Fallback:** Если ключ не задан — используется правиловая система

## Примечание о HH.ru

HeadHunter прекратил поддержку API для соискателей 15 декабря 2025 года.
Начальная версия агента использовала HH.ru API, но после его закрытия был выбран Stepik.
Это решение логично: студентам нужны не только стажировки, но и курсы для подготовки к ним.
